# Fraga Ventures — Admin Dashboard

Internal ops dashboard for Fraga Ventures. Tracks SaaS health, deal pipeline, entity structure, cash position, and content operations.

## Stack

- React Router v7 (SSR)
- TailwindCSS v4
- Supabase (auth + database)
- Deployed on Vercel

## Dev

```bash
npm install
npm run dev
```

Runs at `http://localhost:5173`.

## Env

```
SUPABASE_URL=
SUPABASE_ANON_KEY=
```

## Build

```bash
npm run build
```

- AI Scraping Business for sale etc, therefore providing faster deal research.
